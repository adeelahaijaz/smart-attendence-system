PROJECT: SMART ATTENDANCE

Biometric AI Face Recognition System

⚠️ SERVER ENVIRONMENT NOTE:
This project is built using PHP & MySQL. GitHub Pages only supports static hosting. To run the full system with Database connectivity, please host these files on XAMPP or a PHP Server.

Project Components (PHP Source):

  • device.php - AI Facial Recognition Scanner
  • register.php - Student/Employee Registration
  • get_all_faces.php - Database API
  • mark_attendance.php - Logic for Database entries
